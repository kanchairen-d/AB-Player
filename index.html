⚡ AB Player

B站 / A站 · TVBox 代理 · Docker 即用

🐍 Python 3.12 ⚡ FastAPI ☕ Java Proxy

📺 功能

  • Bilibili（B站）视频解析与代理播放
  • AcFun（A站）视频解析与代理播放
  • TVBox & 影视TV 接口格式
  • M3U 播放列表输出
  • 网页播放器 + 点播 / 直播
  • 后台管理界面

🚀 快速部署

一行命令启动：

docker run -d --name ab-player -p 5081:8080 \
-v /path/to/data:/app/data \
ghcr.io/kanchairen-d/ab-player:latest

或在 compose 里用：

services:
ab-player:
image: ghcr.io/kanchairen-d/ab-player:latest
ports:
- "5081:8080"
restart: unless-stopped

📂 项目结构

  • app/ — Python 后端（FastAPI）
  • app/ABPlayerProxy.java — Java 代理核心
  • data/ — 配置与数据目录
  • Dockerfile — 多阶段构建
